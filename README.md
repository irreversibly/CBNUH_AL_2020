## 전반적인 실습 스케쥴은 실습책자를 참고
## 다음 활동에 대해서는 '학습자료' 폴더의 해당 항목 파일을 참고해주세요.  

1. **알레르기 개론 오리엔테이션** (담당: 강민규 / 월요일 오전 8시-9시 / 알레르기내과 외래 또는 교수연구실(서관 931호)) 
2. **피부반응검사/기관지유발검사 실습** (담당: 고민정간호사, 월요일 오전 9~12시, 장소: 피부반응검사실)
3. **알레르기 환자 case 제출** (이메일 제출)
4. **피부반응검사/기관지유발검사 결과 및 해석 제출** (담당: 강민규 / irreversibly@gmail )
4. **흡입제 환자교육** (담당: 강민규 / 화요일 오전 8시 30분-9시 / 교육 전 '흡입제_사용법_동영상' 폴더의 동영상 미리 시청 -> 교육 후 demo 기계로 자가 사용 영상 녹화 후 이메일 송부 / 교수연구실(서관 931호))
6. **약물알레르기 및 약물부작용 관련 교육** (담당: 지역의약품안전센터 직원 / 수요일 오후 1시 30분~3시 / 알레르기 외래) 
7. **알레르기 환경관리 발표** (담당: 강민규 / 목요일 오후 4-5시 또는 금요일 오후 4-5시 / 교수연구실(서관 931호) / 집먼지진드기, 개 고양이, 꽃가루, 곰팡이, 여름철 환경관리 등 주제를 조원이 겹치지 않게 1가지씩 정해서 PPT로 10분 이내 발표)

## 과제물

1. 과제물은 금요일 저녁까지 'CBNU_2019_실습조_이름_과제이름' 형식에 맞춰 이메일(irreversibly@gmail.com)로 보내주세요.  

- 알레르기 환자 case 자료 (워드 또는 한글파일)
- 기관지유발검사/피부반응검사 결과 해석자료 (워드 또는 한글파일)
- 흡입제 자가 사용 동영상 (turbuhaler, diskus, MDI 각각 1개씩 촬영 / 한번은 설명 없이 전체 과정 동영상을 찍고, 다음에는 매 step을 설명하면서 촬용) 
- 환자 천식조절도평가(asthma control test) 설문 및 두드러기 조절정도(urticaria activity score) 설문 각각 10명 이상 (환자이 설문받고 설문지에 환자이름, 설문받은 학생 이름 적은 다음에 환자 전달)
